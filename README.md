# 《部落好心人》V2｜族語互動偵探遊戲

《部落好心人》是一款可直接在瀏覽器遊玩的純前端教學遊戲。玩家要從六名學生手上的角色卡與交通工具卡取得線索，找出任務指定的答案。

## 線上試玩

- [第一版線上試玩](https://captain-balung.github.io/tribal-kindness-detective/)
- [第二版線上試玩](https://captain-balung.github.io/tribal-kindness-detective-v2/)

## 內容狀態

- 使用語言：秀姑巒阿美語／華語。
- 第二版的族語與文化內容已由族語老師審閱確認。

## 遊玩方式

1. 抽牌後，每名學生會拿到一張角色卡與一張交通工具卡。
2. 抽出任務，取得性別與交通工具兩條線索。
3. 選擇學生，把詞彙拖曳或點選到問題區，調整成想問的句子，再按「提出問題」。
4. 根據學生回答的角色姓名與交通方式進行推理。
5. 進入猜答案模式，找出符合任務條件的學生。

遊戲支援桌面與手機瀏覽器，可使用滑鼠、鍵盤或觸控操作。
